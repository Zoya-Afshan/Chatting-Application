# Project Overview
This is a real-time chatting application built using Java (Swing & Networking APIs).
It allows two users to communicate with each other over a network using Socket Programming.
The project also features a simple graphical user interface (GUI) for sending and receiving messages.

# Features

 Graphical User Interface using Java Swing , 
 Client-Server Communication using Socket and ServerSocket , 
 Timestamps with each message using SimpleDateFormat , 
 Message Formatting with custom colors and borders , 
 Lightweight and beginner-friendly
